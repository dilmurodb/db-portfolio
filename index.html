Крестики-Нолики

Один из первых проектов над которым я работал был создан с помощью HTML, CSS и JavaScript. Проект называется Крестики-нолики, суть игры заключается в том чтобы первым отметить три квадратика непрерывно (по горизонтали, по вертикали или по диогонали). В игре участвуют два игрока, один игрок отмечает квадратики красным а другой синим. Каждый игрок для того чтобы выиграть должен одновременно пресикать возможность другого отметить три квадратика последовательно в то же время создавая удобную позицию для себя.

Тривиа игра

Следующий проект созданный мной называется Тривиа Игра. В этом проекте так же использованы язык программирования JavaScript, HTML и CSS. Суть игры заключается в том чтобы ответить на вопросы правильно и набрать максимальное количество баллов. Для того чтобы начать игру надо выбрать тему Тривиа из следующих вариантов: Кино, Музыка или География.
