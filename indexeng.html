    Tic-Tac-Toe

    One of the first projects I worked on was created using HTML, CSS and JavaScript. The project is called Tic Tac Toe, the essence of the game is to be the first to mark three squares continuously (horizontally, vertically or diagonally). The game involves two players, one player marks the squares in red and the other in blue. Each player, in order to win, must simultaneously prevent the other from marking three boxes in succession while at the same time creating a comfortable position for himself.

    Trivia

    The next project I created is called Trivia Game. This project also uses the programming language JavaScript, HTML and CSS. The essence of the game is to answer the questions correctly and score the maximum number of points. In order to start the game, you need to select a Trivia theme from the following options: Cinema, Music or Geography.
